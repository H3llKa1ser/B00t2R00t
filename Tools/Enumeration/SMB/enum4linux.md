# Enum4Linux SMB Enumeration 

## Flags:

### -u = Get userlist

### -m = Get machine list

### -N = Get namelist dump

### -s = Get sharelist

### -P = Get password policy information

### -G = Get group and member list

### -a = ALL OF THE ABOVE (Basic enumeration)

### smbclient \\IP\SHARE -U USER
