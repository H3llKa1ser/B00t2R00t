## Commands

#### net user /domain

#### net user USER /domain

#### net group /domain

#### net group GROUP /domain

#### net accounts /domain

## BENEFITS

#### 1) No additional or external tooling required

#### 2) NO GUI

#### 3) Can be used with phishing payloads

## DRAWBACKS

#### 1) Must be executed from a domain-joined machine

#### 2) May not show all information
